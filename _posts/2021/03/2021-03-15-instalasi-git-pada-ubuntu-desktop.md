### Install GIT

Lakukan perintah ini pada terminal.

```bash
sudo apt update
sudo apt-get install git
```

### Konfigurasi GIT

```bash
git config --global user.name "letakan_username_disini"
git config --global user.email "letakan_email_disini"
git config --global credential.helper store
```
