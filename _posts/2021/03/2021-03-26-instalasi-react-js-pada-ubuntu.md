## Persiapan

```bash
sudo apt-get update && apt-get upgrade
```

## Instalasi NPM

```bash
sudo apt install npm
npm --version
node --version
```

## Instalasi Create-React-App (CRA)

```bash
sudo npm -g install create-react-app
create-react-app --version
```
